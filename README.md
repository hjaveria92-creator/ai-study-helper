# AI Study Helper

## Deployment Steps
1. **Create GitHub account**: If you don't already have one, visit [GitHub](https://github.com) to create an account.
2. **Upload project**: After creating your account, upload the project using the GitHub interface or command line.
3. **Go to Vercel**: Visit [Vercel](https://vercel.com) to access the platform for deploying your project.
4. **Import project**: Use the ‘Import Project’ option on Vercel and select your GitHub repository.
5. **Add ENV variable**: Set the environment variable `OPENAI_API_KEY=yourkey` in the settings for your Vercel deployment.
